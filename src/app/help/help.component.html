This page aims to help users solve problems when they encounter difficulties. This page also includes detailed instructions on how to use this app. The upper part of this page is the navigation bar, which you can switch pages by clicking on. Then there are several features of this app:

  • Add Artist: Fill in the form content and click the add button to add an artist. The table below can be displayed in real-time. Attention: The correct information must be added, with the ID being a number and the rest being a string. And special notes can be filled in according to the needs to determine whether it is a featured artist (if there is information on special notes, it is a featured artist).
  • Edit Artist: By editing the artist information in the form and clicking the edit button, submit the modified information. The table below can also display the modified information in real time.
  • Delete Artist: Delete artist by searching for artist ID.
  • Search Artist: Use artist ID to search for artists.
  • Featured Artists: Special notes can be filled out according to needs to determine whether one is a featured artist (if there is information on the special notes, it is a featured artist), and the table below displays it in real-time.
Error Messages:
  • "An artist with the same ID already exists.": The ID already exists in the database, but the user entered a duplicate ID.
  • "Artist name must be entered.": The user may have forgotten to enter the name.
  • "DOB must be entered.": Can the user forget to enter DOB.
  • "Contact information must be entered.": Users may have forgotten to enter the contact information.
  • "Exhibition date must be entered.": Users may have forgotten to enter the Exhibition date.
  • "Not Found Artist!": No artist found.
  • "The artist list is empty.": The list of stored artists is empty.
  • "Unable to find the artist ID to delete.": The artist ID provided for deletion cannot be found.
